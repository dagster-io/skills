# dg api schedule get

```bash
dg api schedule get <SCHEDULE_NAME>
```

## General

Retrieve details for a specific schedule by name from Dagster Plus.

For shared flags (`--json`, `--response-schema`, `--deployment`, `--organization`, `--api-token`, `--view-graphql`), see [general.md](../general.md).
