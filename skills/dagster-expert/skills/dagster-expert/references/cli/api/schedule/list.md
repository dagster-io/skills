# dg api schedule list

```bash
dg api schedule list
```

## General

List all schedules from Dagster Plus.

For shared flags (`--json`, `--response-schema`, `--deployment`, `--organization`, `--api-token`, `--view-graphql`), see [general.md](../general.md).

## --status

Filter schedules by status (e.g. RUNNING, STOPPED).
