# dg api agent get

```bash
dg api agent get <AGENT_ID>
```

## General

Retrieve details for a specific agent by ID from Dagster Plus.

For shared flags (`--json`, `--response-schema`, `--deployment`, `--organization`, `--api-token`, `--view-graphql`), see [general.md](../general.md).
