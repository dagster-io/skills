# dg api sensor list

```bash
dg api sensor list
```

## General

List all sensors from Dagster Plus.

For shared flags (`--json`, `--response-schema`, `--deployment`, `--organization`, `--api-token`, `--view-graphql`), see [general.md](../general.md).

## --status

Filter sensors by status (e.g. RUNNING, STOPPED).
