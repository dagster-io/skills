# CLI Reference

The `dg` CLI provides commands for scaffolding, launching, inspecting, and validating Dagster projects.

- [scaffold](./scaffold.md) - Create definitions (assets, schedules, sensors, components)
- [launch](./launch.md) - Materialize assets or execute jobs
- [list](./list.md) - Discover and inspect definitions
- [check](./check.md) - Validate project configuration
- [asset-selection](./asset-selection.md) - Asset selection syntax
- [api](./api.md) - API access and Dagster Plus authentication
